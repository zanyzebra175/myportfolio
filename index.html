Overview:

The Samson Realty Project was a redesign of a realty company website in Providence, Rhode Island. The old website was outdated, riddled with ads, and not responsive to different devices. This project gave me the opportunity to practice both simplistic design and programming skills, as I created a much more simplistic interface for the realty company that was responsive to multiple device sizes. You can read more about this project by clicking this link.

Goods:

This project gave me the opportunity to work heavily with wireframing, specifically through Figma. In redesigning the website from scratch, I had to take into account design principles such as color, layout, and accessibility. Additionally, I had the opportunity to work with the Bootstrap library in my web development for this project, giving me experience with important web elements such as Nav Bars and more.

Growths:

Given the scope of the project, there were elements of the redesigned website that were static/not made to be completely functional. If I had more time, for example, I would have implemented a working carousel feature for the house listings as is demonstrated on the website. However, you may notice that working carousel functionality has been implemented through this portfolio, so I did have the opportunity to work on ideas for growth and implement them going forward!
